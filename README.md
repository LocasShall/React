canvas/数据可视化

 - [Chart.js][30] 使用`<canvas>`标签的简易HTML5图表。`star:14600`

模块管理/加载器
--------

 - [ESL][34]   是一个浏览器端、符合`AMD`的标准加载器，适合用于现代`Web`浏览器端应用的入口与模块管理。

`ESL` vs `RequireJS`

    体积更小 (Smaller)
    性能更高 (Higher performance)
    更健壮 (More Robustness)
    不支持在非浏览器端使用 (Browser only)
    依赖模块用时定义 (Lazy define)

 - [seajs][35] 一个遵循`CommonJS`规范的`JavaScript`模块加载器。提供简单、极致的模块化开发体验。`star:4100`

 - [Component][36] 一个模块化的`JavaScript`框架，同时也是面向前端的包管理器。

动画
--

 - [animate.css][37] 一个跨浏览器的`CSS`动画库。简单易用易上手。`star:23000`

 - [move.js][38] 极小的 `JavaScript` 库,支持 `CSS3` 的动画效果,非常简单优雅。`star：2600`

 - [TweenJS][39] 是一个简单但强大的 `Javascript` 动画库。`CreateJS` 套件的一部分。`star:1500`

 - [bounce.js][40] 一个用于制作漂亮的 `CSS3` 关键帧动画的 `JavaScript`
   库,使用其特有的方式生成的动画效果。`star:3600`

 - [Swipe][41] 号称最精确的`Slider`触摸库，专为移动设备优化。`star:4.7K`

插件
--

 - [zepto.fullpage][42] 专注于移动端的全屏滚动插件。`star:510`
 
 - [fullPage.js][43] pc端的全屏滚动插件。 `star:9500` [view demo][44] 

 - [onepage-scroll][45] 可以轻松建立一个动感的响应式的滚动效果页面，比较适用于单页面的专题站。支持现代浏览器和IE8以上版本。[View demo][46] 。`star:7700`

 - [slick][47] 一款完全响应式的 `jQuery` 图片滚动插件，能够根据容器自动适应宽度。`star:10000` [view demo][48]

 - [superslides][49] 致力于解决网站大部分特效展示问题。网站上常用的“焦点图/幻灯片”“Tab标签切换”“图片滚动”“无缝滚动”等只需要一个`SuperSlide`即可解决！ [view demo][50] `star:1100` 

 - [jQuery-One-Page-Nav][51] 当用户滚动网页时，实现平滑滚动和智能导航。`star:1100`

 - [slider][52] 一个`jquery`完全开源的`JavaScript`代码库，用户可以开发，调试和深度定制自己的滑块。`star:850` [view demo][53]

框架、库和组件
-----

 - [polymer][54] `web`组件构建框架。一套以“一切皆组件、最少化代码量、最少框架限制”为设计理念的`Web UI`框架。 `star:9900`

 - [meteor][55] 一个超简单的纯`javascript` web框架。用于自动化和简化实时运行的 Web 应用程序的开发。可直接在浏览器或者云平台中运行。 `star:25300`

 - [impress.js][56] 创建令人兴奋的演示。使用`CSS3`的转换和过渡，这个库允许你创建令人印象深刻的演示文稿。[view demo][57]。`star:24300`

 - [ionic][58] 先进的`HTML5` 移动端开发框架。帮助开发者使用HTML5, CSS3和`js`做出不可思议的`hybrid app`。`star:17000`



 - [reveal.js][59] 基于`CSS3`的3D幻灯片工具。能够制作绚丽的演示文稿并生成`HTML`格式，将它发布到web上。`star:21500` [view demo][60]

 - [Modernizr][61] 用来检测浏览器功能支持情况的`JavaScript`库,可以检测18项`CSS3`功能以及40多项关于`HTML5`的功能。`star:16000`

 - [pure.css][62] 一组很小的，响应式的`css`组件，你可以在网页的项目上到处使用。`star:12000`

 - [three.js][63] 是`JavaScript`编写的`WebGL`第三方库。提供了非常多的3D显示功能。`star:20000`

 - [TimelineJS][64] 轻松制作时间轴。`star:8000`

 - [jquery-pjax][65] 对`ajax` 和 `pushState`的封装，让你可以很方便的使用`pushState`技术，用以实现页面无刷新加载。`star:11500`

 - [highlight.js][66] `javascript`语法高亮。既可以运行在浏览器端也可以运行在服务端。`star:5500`

 - [commander.js][67] `Node.js`命令行工具。`star:3800`

 - [togetherjs][68] 由`Mozilla`打造的一款可以给网站添加实时协作功能的`JavaScript`库。`star:5K`

 - [HTML.js][69] 轻量级的简化与`DOM`操作的js库。[view demo][70] `star:1.5K`

 - [MEAN.JS][71] 全栈式javascript，使用`MongoDB, Express,
   AngularJS`和`Node.js`。`star:2.2K`

 - [wechat.js][72] 微信相关的 js 操作：分享、网络、菜单。`star:700`

 - [JavaScript-Load-Image][73] 一个`js`加载和转换图像文件的库。 `star:1.2K` [view
   demo][74]

 - [progress.js][75] 一个 `js` 和 `CSS3`的库，帮助开发人员为网页上的每个对象创建和管理进度条效果。`star:1.6K` [view demo][76]

 - [foundation][77] 号称世界上最先进的响应式前端框架，也是一款`Mobile First`的框架。`star:21K`

 - [Sugar][78] 一个`JavaScript`库。它扩展了现有的`JS`对象的方法，让你可以用更少的代码做更多的事情。`star:2.8K`
